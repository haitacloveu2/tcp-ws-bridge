# tcp-ws-bridge
Forward TCP to Websocket

# or install globally

npm install -g tcp-ws-bridge tcp-ws-bridge

Options:

  --thost  address tcp.     [required]

  --tport  port tcp.        [required]

  --wport  port websocket.  [required]

example:

$ tcp-ws-bridge --thost=127.0.0.1 --tport=5000  --wport=5001
